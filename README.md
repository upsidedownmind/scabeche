scabeche
========

Simple cache in scala (based on memcached)


Start Server
-----------

Start sbt: 
	> run

Port is 5544.

Usage
-----

Just telnet to localhost 5544

Commands
--------

    set <key> <value>\r\n
    get <key>\r\n
    remove <key>\r\n
    list\r\n
    size\r\n
    quit\r\n

Enjoy!
